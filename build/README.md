# Build resources

Drop your app icons here:

- `icon.ico` — Windows installer + window icon (256x256 multi-res preferred)
- `icon.icns` — macOS .dmg + dock icon
- `icon.png` — Linux + fallback (512x512 recommended)
- `tray-icon.png` — system tray icon (16x16 or 32x32)

Until you add real icons, electron-builder will fall back to defaults.
